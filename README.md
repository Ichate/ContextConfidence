# ContextConfidence
This is my implementation of flask and transformers library, I use deepset/roberta-base-squad2 for this project.

# Setup
You need 2 libraries for this project
* flask
* transformers
to install do:
```
pip install flask transformers
```
# Running
To run this just do:
```
python app.py
```
